# MyWalletTool - Web3钱包工具

## 功能特性

- 生成随机EVM地址
- 生成随机Solana地址
- 批量创建新钱包（包含助记词）
- 链上转账功能（支持EVM链和Solana链）
- 图形化界面（GUI）支持
- config的okx文本包含主流链的usdt和usdc合约

## 小白直接下载release运行

## 程序员clone后运行以下

```bash
cd /MyWalletTool
pip install -r requirements.txt
python wallet_gui.py
```

PS：exe 同目录下会生成 logs 文件夹，链上打新、mint 等功能等待后续开发。
